# Extended Ascii C.
### Extended Ascii table created with C. Showing characters from 0 to 255 (undisplayable characters will be printed with "**` `**").
##### Compile the **`Main.c`** with **`gcc`** (command: **`gcc -fdiagnostics-color=always -g Main.c -o name-to-executable -Wall -Wextra -O3 -std=c23`**).
